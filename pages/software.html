xlogit: An Open-Source Python Package for GPU-Accelerated Estimation of Mixed Logit Models.

xlogit offers a comprehensive set of tools for the estimation of Mixed Logit models, along with extra functionalities to estimate Multinomial and Conditional Logit models. xlogit's main feature is the use of GPU acceleration to significantly reduce estimation times. On average, xlogit is 48 times faster than existing estimation tools even using a low-end graphics card. For model specification, xlogit allows the incorporation of random parameters with several types of mixing distributions (normal, lognormal, triangular, uniform, and truncated normal). Also, it supports panel data and offers post estimation tools for prediction and specification testing. (URL: https://github.com/arteagac/xlogit)
