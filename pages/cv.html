ACHIEVEMENTS

  • Created xlogit, an open-source software for accelerated estimation of Mixed Logit models that is, on average, 48 times faster than existing implementations and can escalate to model estimation using millions of random draws. A paper describing xlogit was recently accepted in the Journal of Choice Modeling (https://doi.org/10.1061/9780784482865.024)
  • Developed an approach to assist analysts in the selection of a Mixed Logit model specification. The approach strategically tests multiple combinations of covariates and random parameter distributions using an optimization metaheuristic to find the model specification that offers the best fit to the data. A paper describing the approach is published in the Journal of Choice Modeling (https://doi.org/10.1016/j.jocm.2019.01.001)
  • Developed an approach that massively analyzes traffic crash narratives using Natural Language Processing to better understand the factors that increase the severity of traffic crashes. OPEN-SOURCE SOFTWARE

  • xlogit, An open-source Python package for GPU-accelerated estimation of mixed logit models: xlogit offers a comprehensive set of tools for the estimation of Mixed Logit models, along with extra functionalities to estimate Multinomial and Conditional Logit models. xlogit's main feature is the use of GPU acceleration to significantly reduce estimation times. On average, xlogit is 48 times faster than existing estimation tools even using a low-end graphics card. For model specification, xlogit allows the incorporation of random parameters with several types of mixing distributions (normal, lognormal, triangular, uniform, and truncated normal). Also, it supports panel data and offers post-estimation tools for prediction and specification testing. (URL: https://github.com/arteagac/xlogit)
